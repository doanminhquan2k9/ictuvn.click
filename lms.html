Nếu gặp lỗi khi chạy tool, vui lòng:

  • Tải và chạy file setup.bat để tự động cài đặt thư viện
  • Cài đặt Python 3.12 nếu tool yêu cầu
  • Đảm bảo đã tắt Windows Defender hoặc phần mềm diệt virus
  • Chạy tool với quyền Administrator (nhấn chuột phải chọn "Run as administrator")
  • Kiểm tra kết nối Internet ổn định
  • Khởi động lại máy tính nếu cần thiết

Ctrl + mũi tên lên - Ẩn phần mềm

Ctrl + M - Giải bài tập

Ctrl + Shift - Xem lịch sử hoạt động

Nút "Đóng" - Tắt phần mềm

Alt + F4 - Đóng nhanh ứng dụng

Khắc phục sự cố thường gặp:

  • Nếu tool không chạy, thử tải lại file hoặc kiểm tra kết nối Internet
  • Windows Defender có thể chặn file, cần tắt tạm thời hoặc thêm vào danh sách loại trừ
  • Khởi động lại máy tính nếu gặp lỗi không xác định
  • Đảm bảo đã tải đủ file (kiểm tra dung lượng file)
  • Chạy với quyền admin nếu yêu cầu quyền hệ thống
